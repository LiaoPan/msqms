Installation with pip
---------------------

You can install `OPMQC` by PyPI:

.. code-block:: bash

   pip install opmqc

You can update the latest OPMQC:

.. code-block:: bash

    pip install -U opmqc


Installation from source code:

.. code-block:: bash

    pip install git+https://github.com/liaopan/opmqc
